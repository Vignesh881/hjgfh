MoiBook2025 - Portable Package
==============================

இந்த package-ஐ வேறு laptop-க்கு எளிதாக install செய்யலாம்.

Installation Steps (புதிய launcher):
-----------------------------------
1. முழு 'MoiBook2025_Portable' folder-ஐ copy பண்ணுங்க
2. புதிய laptop-ல் paste பண்ணுங்க (Example: C:\MoiBook2025\)
3. Python install இருக்கா check பண்ணுங்க: python --version
4. START_MOIBOOK_APP.bat double-click பண்ணுங்க
5. உலாவி தானாகவே `http://localhost:3000/` திறக்கும் (server window பெயர் MOIBOOK_STATIC)

Requirements:
------------
- Windows 10 or higher
- Python 3.7 or higher
- 500 MB free space

--------------

- INSTALLATION_GUIDE.md - Complete installation guide
