## ЁЯОп Tamil Shortcut Design Principles

### How Tamil Shortcuts Were Chosen:

1. **First syllable(s)** of town name:
   - роХрпЛропроорпНрокрпБродрпНродрпВро░рпН тЖТ `роХрпЛ`
   - роЪрпЖройрпНройрпИ тЖТ `роЪрпЗ`
   - роородрпБро░рпИ тЖТ `роо`

2. **Distinctive characters** for uniqueness:
   - родро┐рогрпНроЯрпБроХрпНроХро▓рпН тЖТ `родро┐ройрпН` (not just `родро┐` which is родро┐ро░рпБроЪрпНроЪро┐)
   - роХро╛роЮрпНроЪро┐рокрпБро░роорпН тЖТ `роХро╛роЮрпН` (distinctive)

3. **Short and easy** to type:
   - Usually 1-2 characters
   - Common letters used frequently

4. **No ambiguity** between shortcuts:
   - Each shortcut maps to ONE town
   - Similar towns use different patterns
